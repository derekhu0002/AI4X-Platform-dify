# Capability 测试用例：业务上下文归并

## 1. Capability 定义

- 一级能力域：情报接入与标准化
- 关联架构元素：1208、1223、1224、1225、1226、1228、1229
- 覆盖目标：验证系统能否把技术事实映射到业务上下文、资产归属和业务关键度。

## 2. 前置条件

- 先完成 [00-environment-setup.md](./00-environment-setup.md) 中的测试环境搭建，并确认 Dify、OpenCTI、OpenCTI MCP、Notification MCP 均可访问。
- 已导入 VS1、VS3、VS4 数据集。
- `ANALYST_ACCOUNT` 与 `ROLE_SHARED_ACCOUNT` 均可登录。
