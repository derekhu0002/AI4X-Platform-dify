# Capability 测试用例：通知与决策协同

## 1. Capability 定义

- 一级能力域：响应协同与处置编排
- 关联架构元素：1208、1222、1223、1224、1225、1230
- 覆盖目标：验证系统能否把高风险结论、事件结果、漏洞影响或规则上线结果通过标准化通知通道同步给相关角色。

## 2. 前置条件

- 先完成 [00-environment-setup.md](./00-environment-setup.md) 中的测试环境搭建，并确认 Dify、OpenCTI、OpenCTI MCP、Notification MCP 均可访问。
- 已执行 VS1、VS2、VS3、VS4 至少一条主路径。
- `ROLE_SHARED_ACCOUNT` 用于观察业务负责人、安全负责人、管理层、安全运营团队共用视角。
